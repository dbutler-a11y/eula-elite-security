### File Naming Structure
```
[category]_[type]_[variation]_[size].[extension]

Examples:
- character_officer_front_1920x1080.png
- logo_primary_horizontal_gold.svg
- scene_houston_skyline_sunset_4k.jpg
- vehicle_escalade_black_side_hd.png

```

### Version Control
- Use sequential numbers for iterations: `_v1`, `_v2`, `_v3`
- Keep original files with `_original` suffix
- Final approved versions without version suffix
